# Rindy
Ruby Unity Indicator

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rindy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rindy

## Usage

Execute the app by running `bundle exec rindy`
